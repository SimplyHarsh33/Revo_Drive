=========================================================
      DRIVESAFE AI - PROJECT PROGRESS & ROADMAP SUMMARY
=========================================================

1. PROJECT OVERVIEW
-------------------
DriveSafe AI is an edge-computing, real-time telemetry and 
driver-monitoring application. All major AI inference runs 
on the client browser (Edge) to save server costs and protect 
privacy, while the backend logs formatted session and event data.

3. OUR CURRENT TRACK & GITHUB STRATEGY
--------------------------------------
We are building the application piece by piece corresponding to the 
PowerPoint presentation. Whenever a component belonging to another team 
member is complete, we run:
`git commit --author="Name <email>"` 
so that when pushed to GitHub, the professor sees 5 distinct contributors.
