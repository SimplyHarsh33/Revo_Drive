1. PROJECT OVERVIEW
-------------------
DriveSafe AI is an edge-computing, real-time telemetry and 
driver-monitoring application. All major AI inference runs 
on the client browser (Edge GPU) to save server costs and protect 
driver privacy, while the backend API logs formatted telemetry 
events to a robust database schema.
